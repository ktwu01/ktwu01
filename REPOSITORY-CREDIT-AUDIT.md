## Method (how ownership was verified)

2. **Fork filter.** Even under your own account or organization, a repository where GitHub reports `isFork = true` is not original work. Its upstream (`parent`) is recorded for attribution.

Internal forks (a fork whose upstream is one of your own repositories, e.g. `ut01/sean-xiang-chronicle` from `ktwu01/sean-xiang-chronicle`) are flagged separately. They re-host your own work but, under the strict fork rule you chose, are not listed as distinct original creations.

The GitHub API owner field and fork flag are authoritative. Repository name or local folder location is not used to decide credit.
